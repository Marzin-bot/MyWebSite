Marzin Samuel

Présentation

Je m'appelle Marzin Samuel, j'ai actuellement 18 ans et je m'intéresse beaucoup au développement Web et au développement de Jeux vidéo en multijoueur en ligne. Plus tard je voudrais devenir développeur web et/ou de jeu vidéo.

Quelle représentation vous faites vous du métier de développeur informatique ?

Pour moi être développeur informatique c'est pouvoir créer des sites web, créer des applications informatiques, des logiciels, programmer des micros contrôleur. Dans l'absolu être développeur informatique vise à résoudre des problèmes par le code, savoir travailler en équipe et mettre à jour/corriger du code défectueux. Être développeur informatique c'est aussi savoir se documenter pour rester à jour des dernières nouveautés pour toujours proposer le meilleur de l'industrie aux clients. Il faut avoir un bon sens de la logique et décomposer les problèmes en plusieurs sous-parties pour mieux les résoudre et prendre compte de ses erreurs passées pour s'améliorer ensuite. Un bon développeur est un développeur qui maîtrise des outils adaptés suivant le contexte (éditeur de code, IDE, gestionnaire de projets/version, ordinateur/système d'exploitation) et suivre les exigences d'un cahier des charges.

Quelles sont vos connaissances et expériences en informatique ?

J'estime avoir une TRÈS grande expérience dans les langages informatiques suivants: HTML, CSS, Javascript (front-end), Python, SQL. De plus, je maîtrise couramment le langage Php, C, C++, Javascript(Backend Node Js) et je maîtrise très bien le langage très exotique qu'est le GDscript (langage de programmation spécifique à Godot Engine, un moteur de jeu Open Source 2D/3D). J'ai notamment expérimenté pas mal de choses en GDscript et je me suis amusé à implémenté le protocole HTTP1.1 et le protocole PostgreSQL en me basant sur les spécifications de ces derniers.

Protocole PostgreSQL implémenté en GDscript: https://github.com/Marzin-bot/PostgreSQLClient.
De même pour le protocole HTTP implémenté en GDscript: https://github.com/Marzin-bot/HTTPServer.

Cela m'a permis d'avoir de très bonnes connaissances en protocole réseau. Je sais programmer des sites web (full stack) et créer des sites web responsives, accessible pour les personnes en situation de handicap. Je maîtrise également le référencement SEO. Je programme généralement la partie serveur en python directement via le module python WSGI et/ou avec le framework django. J'utilise généralement PostgreSQL pour créer mes bases de données mais j'utilise aussi SQLite pour les petits projets. Je sais également utiliser MySQL et je sais éditer les bases de données via Phpmyadmin et pgAdmin. J'utilise plusieurs éditeurs de texte pour coder notamment Brackets et CodeBlocks. Et je maîtrise d'autres logiciels comme Blender, Godot Engine, Inkscape, GIMP, Wireshark, LMMS, FileZilla, Wireshark...

En ce moment je crée des jeux vidéo multijoueur via Godot Engine et je développe un logiciel en ligne de web design pour la conception de site internet via une interface graphique et qui vise à simplifier le flux de travail des webmasters pour la conception de sites web notamment sur le design graphique des sites web (toujours au stade de développement).
